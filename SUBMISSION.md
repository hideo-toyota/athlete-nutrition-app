## 実績・活動内容としての説明

Vibe Coding で Athlete Nutrition App という動作プロトタイプを作成しました。React / TypeScript / Vite を使い、競技プロフィール、日別カレンダー、食事プラン、写真、体調ログ、AI Coach Notes、CtoC支援、本番日から逆算する Event Fueling までを1つのアプリとして実装しています。

提出時点で TypeScript の型チェック、production build、Playwright E2E を通しており、単なるアイデアではなく、実際に操作できるプロダクトとして提示できます。

## 応募フォーム用説明文

このプロトタイプは、アスリート向けの食事管理を「栄養素の記録」だけで終わらせず、実際に食べられたか、体調とどう関係しているか、周囲が何を支援すべきかまで扱うアプリです。

陸上、野球、サッカー、バレエ、バスケットボール、ゴルフ、水泳、卓球などの競技プロフィールを選択でき、通常日、練習日、試合前日、試合当日、回復日で栄養ターゲットと見方を切り替えます。食事プランは `未実施 / 一部だけ / 完了 / スキップ` で記録でき、睡眠、疲労、食欲、体重、メモも日別に保存します。

月間カレンダーでは、記録のある日、Day Mode、食事完了率を確認できます。各食事カードには写真も添付できるため、文字入力だけでは残しにくい実際の食事内容も日別ログとして保存できます。

今回さらに、試合・大会・発表会などの「本番日」を登録すると、そこから逆算して Event Fueling / 本番逆算プランを表示できるようにしました。体重あたりの糖質・たんぱく質目標を日別に出し、本番3日前から前日はカーボローディング、当日は本番前補給、翌日は回復栄養として扱います。疲労感、睡眠、食欲も補正に使い、疲労が高い日は糖質下限を少し上げる、食欲が低い日はゼリーやおにぎりなど軽い形に寄せる、といった現場向けの提案にしています。

さらにCtoC要素として、バディ同士で当日の食事実行率・睡眠・疲労を確認できる Buddy Check、直近7日のペア達成率を見られる Pair Score、相手に応援スタンプを送る Cheer Stamps を追加しました。加えて、チームメイトの実例食事ログを参考にできる Team Food Log と、補食・水分・回復の声かけを予約できる Reminder Scheduler も入れ、本人だけで完結せず、仲間と支え合って継続する体験をデモできます。

ルールベースの AI Coach Notes では、糖質不足、補食抜け、睡眠不足、実行率低下などを検出し、次に取るべき行動を短く提示します。また、保護者やコーチに送れる共有メモを自動生成し、現場で支援しやすい形にしています。

2日MVPとして React / TypeScript / Vite で実装し、localStorage による日別保存、破損データ保護、E2E 検証まで行っています。

## 差別化ポイント

一般的な食事管理アプリは、入力した栄養素やカロリーの集計に寄りがちです。このアプリでは、現場で続けるために必要な「日々の実行」「体調との関係」「代替案」「周囲への共有」を中心にしています。

特に、学生アスリートでは本人だけでなく保護者、コーチ、チームメイトの関与が大きいため、共有メモ、バディ機能、チーム内フードログ、リマインド予約で「支え合って続ける」設計にしています。

また、試合日が近づいた時に「今日は何gの糖質を目標にすればいいか」「疲れている時は何を優先すべきか」を1日単位で出せる点を強化しました。ACSM / Academy of Nutrition and Dietetics / Dietitians of Canada のスポーツ栄養ガイドラインをベースに、7-12g/kg/day のカーボローディング目安や回復期の糖質・たんぱく質補給を、競技・本番時間・体調ログに合わせて簡易計算します。
